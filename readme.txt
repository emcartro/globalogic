This is a Spring Boot application for managing users, including functionalities for user sign-up and login.

## Prerequisites

1. Before running the application, ensure you have the following installed:

- Java 11 or higher
- Gradle 7.x or higher

2. Build the Project
Run the following command to build the project: ./gradlew build

3. Run the Application
Start the application using the following command: ./gradlew bootRun
The application will be available at http://localhost:7001

4. Test the endpoints using the postman collection
   resources/Globalogic.postman_collection.json
